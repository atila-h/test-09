"Full stack" is a term in software development that refers to the ability and experience in developing both main parts of a website or application, namely the front-end and back-end.

  • UI/UX Designer
  • Web Designer
  • Freelancer

Frontend Frontend refers to the parts of a website that users can directly interact with. In simpler terms, frontend is what the user sees and interacts with.

Frontend technologies include:

  • HTML (Website structure)
  • CSS (Appearance, design, and styling)
  • JavaScript (Interactions and behaviors on the web)
  • Frameworks and libraries such as React, Angular, Vue.js

Backend refers to the parts of a website that operate behind the scenes. This section is responsible for storing data, processing requests, managing databases, and performing tasks essential for the websiteâ€™s proper functionality, but the user does not interact with it directly.

Backend technologies include:

  • Programming languages like Python, JavaScript (Node.js), Ruby, PHP, Java, C#
  • Frameworks such as Django (Python), Express (Node.js), Ruby on Rails, Laravel (PHP)
  • Databases like MySQL, PostgreSQL, MongoDB, SQLite

A Full Stack Developer must have the ability to integrate frontend and backend. This involves working with APIs (Application Programming Interfaces) and coordinating between the client (frontend) and the server (backend).

  • Sver and Web Hosting Managementer (e.g., AWS, Azure, Docker)
  • Version Control using tools like Git and GitHub
  • Software Testing to ensure the system functions correctly
  • Agile Development for managing team projects
